What is Report Master?

Report Master is a robust moderation tool for Question2Answer, designed to enhance or replace the default flagging system

With Report Master, users can easily submit detailed reports, specifying issues like spam, abuse, off-topic content, or inappropriate behavior. Moderators can review reports, suspend users if necessary, and leave notes for team collaboration. All while maintaining full traceability in the Reports Queue.

Key Features

  • Comprehensive Reporting
    Users can report questions, answers, comments, user profiles, private messages, and wall posts.
  • User Suspension
    Moderators can suspend users, with automatic prevention of suspension stacking.
  • Suspension Visibility
    Track user suspensions within the last 24 hours to avoid repeat suspensions.
  • Pending Suspensions
    Suspensions are only triggered when a reported user visits the site again.
  • Restricted Interaction for Suspended Users
    Suspended users can only view content but cannot interact with the site.
  • Report Tracker
    Users can check the status of their reports, whether they’re pending review or resolved.
  • User Report History
    Moderators can view how many times a user has been reported or has reported others to identify malicious behavior.
  • Conversation History Access
    Moderators can temporarily view reported user conversations for context, until the report is resolved.
  • Message Deletion
    Moderators can delete messages as part of managing reports and maintaining community standards.
  • Pupi FNS Integration
    Notifications for solved reports are synced with the Pupi FNS plugin.
