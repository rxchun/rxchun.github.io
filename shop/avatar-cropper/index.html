What is Avatar Cropper?

Q2A Avatar Cropper is a powerful plugin for Question2Answer (Q2A) that allows users to crop and upload square avatars (1:1)

It also includes a batch tool for admins to crop all existing avatars automatically to a centered square format, ensuring visual consistency across your community.

Key Features

  • Live Avatar Cropping
    Users can upload and crop avatars in-browser, with a centered square preview before submission.
  • Modern, Responsive UI
    Styled to match your Q2A theme, includes a background opacity slider, smooth modal, and real-time feedback.
  • Batch Cropping Tool (Admin Panel)
    Instantly crop all existing avatars from the database. Ideal for legacy uploads or switching from rectangular avatars.
  • GDPR-Friendly
    Option to load local JS/CSS resources instead of CDN, compliant with data protection regulations.
  • No Core Hacks Required
    Integrates cleanly via plugin layer, no need to modify core files or themes.

Requirements

  • Q2A version: 1.8+ (older versions untested)
  • PHP GD extension: Required for image processing
  • Avatars stored as blobs in the database for the Batch Crop of existing avatars (not in disk via QA_BLOBS_DIRECTORY)
    You can still upload and crop avatars manually regardless of the storage method.

Why This Plugin?

Most Q2A installations rely on raw avatar uploads, which leads to inconsistent shapes, poor cropping, and alignment issues.

This plugin brings modern avatar handling to Q2A, giving users a clean and professional experience.
